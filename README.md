# Jewellery Shop Management System

## Overview

Jewellery Shop Management System is a web-based academic project developed using PHP and MySQL. The system helps manage inventory, customers, sales, staff records, and business reports through a role-based authentication system.

## Features

### Role-Based Login System

The application provides separate access levels for Admin and Staff users.

### Admin Panel

Admin users have access to:

* Dashboard
* Inventory Management
* Staff Management
* Customer Management
* Sales Management
* Reports Module
* Logout

### Staff Panel

Staff users have access to:

* Dashboard
* Inventory Management
* Customer Management
* Sales Management
* Logout

## Modules

### Dashboard

* Overview of business activities
* Quick access to system modules

### Inventory Management

* Manage jewellery stock
* Add, update, and delete inventory records

### Customer Management

* Maintain customer information
* View customer records

### Sales Management

* Record sales transactions
* Manage sales data

### Staff Management (Admin Only)

* Add and manage staff accounts
* Control user access

### Reports Module (Admin Only)

* View business reports
* Monitor sales and operational data

## Technologies Used

* HTML
* CSS
* JavaScript
* PHP
* MySQL
* phpMyAdmin
* XAMPP

## Installation

1. Clone the repository.
2. Import the provided SQL database file into phpMyAdmin.
3. Place the project folder inside the XAMPP `htdocs` directory.
4. Start Apache and MySQL from XAMPP Control Panel.
5. Open `http://localhost/jsms` in your browser.
